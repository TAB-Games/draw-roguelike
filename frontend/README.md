### QUIZ GAME

A quiz game that was made using React, Redux Saga, Redux Toolkit, and Tailwind CSS

INSTALLS:
npm install --save react-redux redux @reduxjs/toolkit redux-saga
npm install -D tailwindcss
npm install uuid
npm i daisyui
